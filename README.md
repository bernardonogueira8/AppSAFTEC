# AppSAFTEC

Um projeto simples em Flet para automações.

## Descrição

Aplicação desktop/web desenvolvida com Flet para facilitar automações de tarefas. Interface leve, intuitiva e focada em produtividade.

## Funcionalidades

- Interface gráfica com Flet
- Automatização de processos repetitivos
- Controle e monitoramento de ações
- Fácil configuração

## Instalação

1. Crie um ambiente virtual:
    `python -m venv .venv`
2. Ative o ambiente:
    - Windows: `.venv\Scripts\activate`
    - macOS/Linux: `source .venv/bin/activate`
3. Instale dependências:
    `pip install flet`

## Uso

1. Execute o arquivo principal do projeto:
    `python main.py`
2. Acesse a interface exibida pelo Flet.

## Observações

- Personalize automações de acordo com suas necessidades.
- Adicione módulos e integração conforme o projeto evolui.
